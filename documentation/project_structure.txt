Key Implementation Notes:
- Parallelization strategies are in internal/parallel/ (BSP, Pipeline, Work-Stealing)
- The text matching system uses precomputed GloVe embeddings in textmatch/data/
- Unsplash dataset is stored in data/ with processed metadata in metadata/
- Benchmarking infrastructure allows comparing different parallel approaches
